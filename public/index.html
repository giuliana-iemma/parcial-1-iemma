MoviePickr

¿Te ha pasado que encuentras la película perfecta, pero no sabes en qué plataforma está disponible? ¡No te preocupes más!

Hemos creado una API diseñada para que los usuarios puedan explorar fácilmente un amplio catálogo de películas y descubrir en qué plataformas pueden disfrutar de ellas. Simplificamos tu búsqueda para que puedas empezar a ver lo que te gusta, sin complicaciones.

Registrarse

Para poder acceder a la api, deberás registrarte primero.

URL

/users

Method

POST

Body

JSON

{

  • "name": "Tu nombre",
  • "lastname": "Tu apellido",
  • "email": "tuemail@email.com",
  • "password": "Contraseña"

}

Instrucciones - Películas

Ver todas las películas

Recuerda guardar el token obtenido cuando iniciaste sesión para poder acceder al contenido de la API

URL

/movies

Method

GET

Obtener películas en paginación

Para poder ver las películas separadas en grupos en diferentes páginas, deberás proporcionar en tu URL los siguientes datos:

URL

/movies/pagination/{pagina}/{cantidad}

Página

Coloca el número de página que deseas ver

Cantidad

Coloca el número de películas que deseas ver en cada página

Method

GET

Buscar películas por el ID

Para encontrar una película por el ID que tienen asignado en la base de datos deberás usar los siguientes parámetros:

URL

/movies/{id}

ID

Coloca el ID asignado en la base de datos a la película

Method

GET

Buscar películas por el nombre

Para encontrar una película por el su título deberás usar los siguientes parámetros:

URL

/movies/title/{titulo-separado-con-guiones}

Titulo

Coloca el titulo de la película que estás buscando. Recuerda separar las palabras del título con guiones "-"

Method

GET

Buscar películas por género

Para encontrar una películas que posean determinados géneros deberás usar los siguientes parámetros:

URL

/movies/search/genre?genres={Action-Science_Fiction}

Género

Coloca el o los géneros que deseas buscar. Para separar diferentes géneros, usa "-". Para separar palabras dentro de un mismo género usa "_". Cada palabra debe iniciar con mayúscula, como se muestra en el ejemplo

Method

GET

Ver películas ordenadas alfabeticamente

Para ver las películas en orden utiliza los siguientes parámetros:

URL

/movies/sorted/{a}{d}

"A" "a"

Coloca la letra "a" en la URL para ver las películas en orden ascendente, es decir de la A a la Z

"D" "d"

Coloca la letra "d" en la URL para ver las películas en orden descendente, es decir de la Z a la A

Method

GET

Agregar película

Para poder agregar una nueva película, deberás utilizar los siguientes parámetros.

URL

/movies

Method

POST

Body

JSON.

Validaciones

Título: al menos 4 caracteres. Requerido

Descripción: al menos caracteres. Requerido

Genero: Debe ser un array. Debe contener dos claves: "name" de al menos 3 caracteres y "description". Requerido

Plataformas:Debe ser un array. Debe contener dos claves: "name" y "url". La URL debe poseer "http://" como encabezado. Requerido

{

  • "title": "Nuevo título",
  • "description": "Descripción de la película",
  • "genre": [
    { "name": "Action" } ],
  • "platforms": [
    { "name": "Netflix", "url": "http://www.netflix.com" } ]

}

Editar película

Para poder editar una película, deberás utilizar los siguientes parámetros:

URL

/movies/{id}

Method

PUT

Body

JSON. Coloca únicamente los datos que quieras modificar.

{

  • "title": "Nuevo título"

}

Borrar película

URL

/movies/{id}

Method

DELETE
